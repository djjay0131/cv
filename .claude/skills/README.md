# Constellize Skills

Skills auto-load when Claude Code opens a project containing them.

## Memory Skills
| Skill | Purpose |
|-------|---------|
| constellize:memory:establish | Initialize memory bank for a project |
| constellize:memory:update | Keep memory bank current with code changes |
| constellize:memory:revise | Restructure memory bank when files grow unwieldy |
| constellize:memory:recover | Rebuild a neglected memory bank from code |

## Feature Skills
| Skill | Purpose |
|-------|---------|
| constellize:feature:specify | Adversarial interview → feature spec in llm/features/ |
| constellize:feature:implement | Star-gap-generate code + tests from spec |
| constellize:feature:verify | 4 quality gates: tests, health, deploy, maintainability |

## Design Skills
| Skill | Purpose |
|-------|---------|
| design:tui-designer | Design TUI interfaces with Renaissance principles |
| design:tui-reviewer | Audit TUI against Renaissance standards |
